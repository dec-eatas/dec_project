## 画面構成
* 画面デザインは極力シンプルにして、内部処理に集中
* 機能ごとに必要な画面、要素はQiitaとYahoo!知恵袋を参考に(ほぼ丸パクリでもいい)する
* 機能を追加しやすいよう、各機能を一つのブロックとして扱う
* メニュー、サブメニュー(サイドメニュー)、コンテンツの三要素に大きく分割する
